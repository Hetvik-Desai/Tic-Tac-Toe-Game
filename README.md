#Tic-Tac-Toe Game
<br>
<br>
Overview:-
<br>
This project is a beginner-level implementation of the classic Tic Tac Toe game, built using HTML, CSS, and JavaScript. The game allows two players to take turns and play directly in the browser. The purpose of this project was to combine basic front-end structure with interactive functionality using JavaScript.
<br>
<br>
Objective:-
<br>
The main objective of this project was to strengthen understanding of core web development concepts while introducing interactivity. It focuses on handling user input, updating the UI dynamically, and implementing simple game logic. Building a familiar game helped in understanding how logic and design work together in real-world applications.
<br>
<br>
Tech Stack:-
<br>
HTML5 - for structuring the game layout
<br>
CSS3 - for styling the board and improving visual appearance
<br>
JavaScript (Vanilla JS) - for game logic, event handling, and interactivity
<br>
<br>
Features:-
<br>
Interactive 3x3 Tic Tac Toe grid
<br>
Two-player turn-based gameplay (X and O)
<br>
Win detection logic for rows, columns, and diagonals
<br>
Reset/Restart button to start a new game
<br>
Visual feedback for game status (winner or draw)
<br>
<br>
Learning Outcome:-
<br>
Better understanding of DOM manipulation using JavaScript
<br>
Hands-on experience with event listeners and user interaction
<br>
Improved logic-building skills through game development
<br>
Strengthened knowledge of integrating HTML, CSS, and JavaScript in a single project
